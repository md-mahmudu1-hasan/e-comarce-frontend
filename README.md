## 🚀 Features

### 📧 Real-Time Email Notification System
- Integrated Nodemailer  
- Sends order confirmation emails to customers  
- Sends order alerts to admin instantly after purchase  

### 🔍 Backend Search & Product Discovery
- Server-side search functionality  
- Efficient handling of large datasets  
- Fast and optimized response time  

### ⭐ Customer Review & Rating System
- Authenticated users can add reviews and ratings  
- Improves trust, transparency, and user engagement  

### 🛍️ Core eCommerce Functionalities
- Product listing & filtering  
- Add to cart & checkout  
- Order management system  
- Authentication (Firebase)
